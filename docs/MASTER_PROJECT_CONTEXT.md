# MASTER PROJECT CONTEXT – Poker Tournament Director

**Project Overview**  
Full-featured offline Android poker league manager (Kotlin + Jetpack Compose).  
Manages complete seasons + nightly tournaments with clock, blinds, money segmentation (season pool / night pot / bounty), eliminations, standings, payouts, stats, guests, and exports.
